Sokoban

The famous game in a first person 3D view. Move the boxes to the indicated locations. This might be hard without top view, so differing to the original its also possible to pull boxes.

Mazes

Just a maze.

User Controls

Controlling depends on the platform. In general, keys x/y/z can be used to fine tune the cameras position.

Keyboard

Use arrow key for moving.

Touchscreen

Move by touching segments on the screen.

VR

Use left trigger for teleporting to marked element. Use right trigger to do an action like moving a box. Note: Only the trigger button of a controller is used. The left controller has a wristwatch controller with buttons for

  • Move VR position up/down
  • Pulling a box
